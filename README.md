# Лабораторная работа №3

## По дисциплине «Исследование операций»

---

## 📌 Тема

**Модификации симплекс-метода: метод искусственного базиса и метод больших штрафов**

---

## 🎯 Цель работы

Изучение методов искусственного базиса и больших штрафов для решения задач линейного программирования в канонической форме, не подготовленных к применению симплекс-метода в чистом виде, а также получение навыков их программной реализации.

---

## 📋 Задания

### 🔹 Задание 1

Изучить метод искусственного базиса (двухфазный метод) и реализовать алгоритм решения задачи линейного программирования.

✔ Выполнено:

* Реализован двухфазный симплекс-метод на Python
* Построены 13 блок-схем алгоритма
* Выполнено тестирование программы
* Получены результаты решения

---

### 🔹 Задание 2

Изучить метод больших штрафов (Big M) и реализовать соответствующий алгоритм.

✔ Выполнено:

* Реализован метод больших штрафов
* Построены 9 блок-схем алгоритма
* Проведено тестирование
* Получены результаты вычислений

---

### 🔹 Задание 3

Решить одну из задач вручную (вариант №8).

✔ Выполнено:

* Выполнено ручное решение задачи
* Условие задачи представлено на скриншотах 1–2 (см. также файл `uslovie.png`)
